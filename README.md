# Album 63 - Break the altars

Album 63 is a prophetic, hard-edged worship project using intense rock, metal, and folk textures to denounce spiritual idolatry and cultural deception and to call listeners to repentance and Christ-centered renewal. It draws on biblical imagery (altars, prophets, Mishpat/justice) and contemporary cultural references, then redirects the focus to the cross—Christ’s judgment of false gods and restoration of God’s people.
